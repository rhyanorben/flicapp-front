# FlicApp

---

Front-end desenvolvido com Next.js

O FlicAPP é uma plataforma web e mobile que conecta clientes a prestadores de serviços de forma rápida e segura.

O cliente descreve o que precisa e a Inteligência Artificial interpreta a solicitação, identifica a categoria do serviço e localiza prestadores próximos disponíveis.

O usuário escolhe o profissional, efetua o pagamento seguro dentro da plataforma e o prestador executa o serviço, com o pagamento finalizado de forma justa após a conclusão.
